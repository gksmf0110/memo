# markdown

## 1. 개인메모
```
가장 큰 제목 # 제목
두번쨰 큰 글씨 ## 제목2
세번째 글씨 ### 내용1-1
네번째 글씨 #### 내용1-1-1
...

글씨 앞뒤로 __ 붙이면 볼드 스타일  ex) __볼드스타일문자__

글씨 앞뒤로 _ 붙이면 이태릭 스타일  ex) _이태릭스타일_

글씨 앞뒤로 ___ 붙이면 볼드+이태릭  ex)___볼드+이태릭___

글씨 앞뒤로 '' 붙이면 하이라이트 ex) 이 문자는 '하이라이트' 입니다.

줄 내림(Enter)은 스페이스(띄어쓰기) 두칸하고 엔터 
 -> 개요나 그런건 자동 줄내림이 되고, 그냥 일반 텍스트 같은거에 적용

인용구문은 앞에 > 붙이기  ex) > 하늘이 블로그 - http://haneul.blog.com

하이퍼링크 [클릭](이동하고자하는 링크)

이미지 삽입 ![이미지가 깨질때 나오는 text](이미지 주소)
```
